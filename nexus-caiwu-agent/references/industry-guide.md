## 1. 高科技行业

### 行业特征
- **高毛利率**: 40-70%（软件、半导体通常更高）
- **高研发投入**: 研发费用率 5-25%
- **轻资产**: 固定资产占比相对较低
- **高成长性**: 收入和利润增长潜力大
- **技术迭代快**: 产品生命周期短

### 关键财务指标

| 指标 | 行业基准 | 说明 |
|------|----------|------|
| 毛利率 | 40-70% | 反映技术壁垒和定价能力 |
| 净利率 | 10-35% | 盈利能力强 |
| ROE | 10-30% | 权益收益率高 |
| 资产负债率 | 15-50% | 负债率通常较低 |
| 研发费用率 | 5-25% | 核心竞争力指标 |

### 分析要点

#### 1. 研发投入
- 研发费用率 > 15%: 技术投入积极
- 研发费用率 5-15%: 正常水平
- 研发费用率 < 5%: 需关注技术竞争力

#### 2. 技术壁垒
- 专利数量和质量
- 技术人员占比
- 核心技术自主可控性

#### 3. 成长性指标
- 收入增长率（同比、环比）
- 新产品收入占比
- 客户集中度

#### 4. 现金流特征
- 经营现金流通常为正
- 资本开支较大（扩产、研发）
- 关注自由现金流

### 代表企业
- **中芯国际** (688981): 半导体制造
- **寒武纪** (688256): AI芯片
- **金山办公** (688111): 办公软件
- **用友网络** (600588): 企业软件

### 重点关注
1. 技术迭代风险
2. 研发投入产出比
3. 客户粘性和转换成本
4. 国际贸易环境影响

## 2. 制造业

### 行业特征
- **中等毛利率**: 15-30%
- **重资产**: 固定资产占比高
- **周期性**: 受宏观经济影响较大
- **规模效应**: 产量越大成本越低

### 关键财务指标

| 指标 | 行业基准 | 说明 |
|------|----------|------|
| 毛利率 | 15-30% | 成本控制能力 |
| 净利率 | 4-15% | 薄利多销特征 |
| ROE | 6-20% | 杠杆和周转驱动 |
| 资产周转率 | 0.5-1.8 | 资产利用效率 |
| 存货周转率 | 3-10 | 库存管理能力 |

### 分析要点

#### 1. 产能利用率
- 产能利用率 > 85%: 满产，可能需要扩产
- 产能利用率 70-85%: 正常运营
- 产能利用率 < 70%: 产能过剩风险

#### 2. 成本控制
- 原材料价格波动影响
- 规模效应带来的成本优势
- 自动化水平

#### 3. 存货管理
- 存货周转天数
- 存货跌价准备
- 产成品占比

#### 4. 应收账款
- 应收账款周转率
- 账龄结构
- 坏账风险

### 代表企业
- **三一重工** (600031): 工程机械
- **格力电器** (000651): 家电制造
- **福耀玻璃** (600660): 汽车零部件
- **汇川技术** (300124): 工业自动化

### 重点关注
1. 原材料价格波动
2. 产能扩张节奏
3. 海外市场拓展
4. 产业升级和转型

## 3. 建筑行业

### 行业特征
- **低毛利率**: 8-12%（薄利多销）
- **高负债运营**: 资产负债率 70-80% 为常态
- **项目制**: 收入确认周期长
- **现金流波动大**: 回款周期不均

### 关键财务指标

| 指标 | 行业基准 | 说明 |
|------|----------|------|
| 毛利率 | 8-12% | 行业特性，低毛利 |
| 净利率 | 2-4% | 薄利多销 |
| 资产负债率 | 70-80% | 高杠杆运营 |
| ROE | 8-15% | 杠杆拉高 ROE |
| 现金流/净利润 | 0.8-1.2 | 现金流质量 |

### 分析要点

#### 1. 订单储备
- **新签订单**: 未来收入保障
- **在手订单**: 订单/营收比
- **订单结构**: 毛利率水平

#### 2. 现金流管理
- 经营现金流与净利润匹配度
- 应收账款账龄
- 保证金占用

#### 3. 融资成本
- 有息负债规模
- 利息保障倍数
- 融资渠道

#### 4. 项目风险
- 项目进度
- 合同纠纷
- 业主信用

### 代表企业
- **中国建筑** (601668): 综合建筑龙头
- **中国中铁** (601390): 铁路建设
- **中国交建** (601800): 港口公路
- **金螳螂** (002081): 装修装饰

### 重点关注
1. 房地产政策影响
2. 基建投资力度
3. 应收账款回收风险
4. 海外项目风险
