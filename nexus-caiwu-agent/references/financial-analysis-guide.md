## 一、财务报表基础知识

### 1.1 利润表 (Income Statement)

利润表反映企业在一定会计期间的经营成果。

**核心结构**:
```
营业收入
  - 营业成本
  = 毛利润
  - 销售费用
  - 管理费用
  - 财务费用
  - 研发费用
  = 营业利润
  + 营业外收支
  - 所得税
  = 净利润
```

**关键科目解读**:
| 科目 | 含义 | 分析要点 |
|------|------|----------|
| 营业收入 | 主营业务收入 | 增长趋势、季节性、行业地位 |
| 营业成本 | 直接生产成本 | 成本控制能力、毛利率变化 |
| 毛利润 | 营收-成本 | 产品竞争力、定价能力 |
| 期间费用 | 销管财研费用 | 运营效率、管理能力 |
| 营业利润 | 核心经营利润 | 主业盈利能力 |
| 净利润 | 最终利润 | 股东回报、盈利质量 |

### 1.2 资产负债表 (Balance Sheet)

资产负债表反映企业在特定时点的财务状况。

**核心等式**: `资产 = 负债 + 所有者权益`

**资产结构**:
```
流动资产
  - 货币资金
  - 应收账款
  - 存货
  - 其他流动资产

非流动资产
  - 固定资产
  - 无形资产
  - 长期投资
  - 其他非流动资产
```

**负债结构**:
```
流动负债
  - 短期借款
  - 应付账款
  - 预收款项
  - 一年内到期的非流动负债

非流动负债
  - 长期借款
  - 应付债券
  - 递延所得税负债
```

**关键科目解读**:
| 科目 | 含义 | 分析要点 |
|------|------|----------|
| 货币资金 | 现金及等价物 | 流动性、偿债能力 |
| 应收账款 | 未收回的货款 | 回款能力、坏账风险 |
| 存货 | 库存商品/原材料 | 周转效率、积压风险 |
| 固定资产 | 厂房设备等 | 产能、折旧 |
| 商誉 | 并购溢价 | 减值风险 |
| 短期借款 | 一年内到期债务 | 短期偿债压力 |
| 长期借款 | 一年以上债务 | 财务杠杆 |

### 1.3 现金流量表 (Cash Flow Statement)

现金流量表反映企业现金流入流出情况。

**三大活动**:
```
经营活动现金流
  - 销售商品收到现金
  - 购买商品支付现金
  = 经营现金流净额

投资活动现金流
  - 购建固定资产支付
  - 投资支付/收回
  = 投资现金流净额

筹资活动现金流
  - 借款收到/偿还
  - 股利分配
  = 筹资现金流净额
```

**现金流组合分析**:
| 经营 | 投资 | 筹资 | 企业状态 |
|------|------|------|----------|
| + | - | + | 成长期（借钱扩张） |
| + | - | - | 成熟期（还债分红） |
| + | + | - | 衰退期（变卖资产） |
| - | + | + | 困难期（借钱度日） |

## 二、关键财务指标详解

### 2.1 盈利能力指标

#### ROE（净资产收益率）
```
公式: ROE = 净利润 / 净资产
判断标准:
  - > 15%: 优秀
  - 10-15%: 良好
  - 5-10%: 一般
  - < 5%: 较弱
```

**ROE 是最重要的盈利指标**，反映股东权益的回报率。巴菲特认为长期 ROE > 15% 的公司值得投资。

#### ROA（总资产收益率）
```
公式: ROA = 净利润 / 总资产
判断标准:
  - > 6%: 优秀
  - 4-6%: 良好
  - 2-4%: 一般
  - < 2%: 较弱
```

ROA 反映企业运用全部资产获利的能力，不受资本结构影响。

#### 毛利率
```
公式: 毛利率 = (营业收入 - 营业成本) / 营业收入
判断标准: 行业差异大，需与同行对比
  - 高科技: 40-60%
  - 消费品: 30-50%
  - 制造业: 15-30%
  - 建筑业: 5-15%
```

毛利率反映产品定价权和成本控制能力。

#### 净利率
```
公式: 净利率 = 净利润 / 营业收入
判断标准:
  - > 20%: 优秀（如白酒、医药）
  - 10-20%: 良好
  - 5-10%: 一般（如建筑、零售）
  - < 5%: 较弱
```

### 2.2 偿债能力指标

#### 资产负债率
```
公式: 资产负债率 = 总负债 / 总资产
判断标准:
  - < 40%: 低风险
  - 40-60%: 适中
  - 60-70%: 需关注
  - > 70%: 高风险
```

**行业差异**:
- 银行、地产：负债率高（60-90%）正常
- 建筑、基建：负债率高（70-80%）常见
- 消费、科技：负债率低（30-50%）为佳

#### 流动比率
```
公式: 流动比率 = 流动资产 / 流动负债
判断标准:
  - > 2: 健康
  - 1.5-2: 正常
  - 1-1.5: 需关注
  - < 1: 风险（流动资产不足以覆盖短期债务）
```

#### 速动比率
```
公式: 速动比率 = (流动资产 - 存货) / 流动负债
判断标准:
  - > 1: 健康
  - 0.5-1: 正常
  - < 0.5: 需关注
```

速动比率剔除了变现较慢的存货，更能反映真实偿债能力。

### 2.3 运营效率指标

#### 资产周转率
```
公式: 资产周转率 = 营业收入 / 总资产
判断标准: 越高越好，行业差异大
  - 零售: > 2
  - 制造: 0.5-1.5
  - 重资产: 0.3-0.8
```

反映资产利用效率，周转率越高说明资产创造收入能力越强。

#### 存货周转率
```
公式: 存货周转率 = 营业成本 / 平均存货
判断标准: 越高越好
  - > 10: 优秀（快消品）
  - 5-10: 良好
  - 2-5: 一般
  - < 2: 需关注（可能积压）
```

#### 应收账款周转率
```
公式: 应收账款周转率 = 营业收入 / 平均应收账款
判断标准: 越高越好
  - > 12: 优秀（回款快）
  - 6-12: 良好
  - 3-6: 一般
  - < 3: 需关注（回款慢）
```

### 2.4 现金流指标

#### 现金流质量
```
公式: 现金流质量 = 经营现金流净额 / 净利润
判断标准:
  - > 1.2: 优秀（盈利质量高）
  - 1-1.2: 良好
  - 0.8-1: 需关注
  - < 0.8: 风险（利润含金量低）
```

**现金流质量 < 1 意味着**：
- 利润是"账面利润"，没有真金白银
- 可能有大量应收账款
- 可能存在财务造假风险

#### 自由现金流 (FCF)
```
公式: 自由现金流 = 经营现金流 - 资本支出
判断标准:
  - 持续为正: 优秀（有分红/扩张潜力）
  - 偶尔为负: 正常（扩张期）
  - 持续为负: 需关注（需要持续融资）
```

## 三、杜邦分析法

### 3.1 杜邦公式
```
ROE = 净利率 × 总资产周转率 × 权益乘数

其中:
  净利率 = 净利润 / 营业收入        → 盈利能力
  总资产周转率 = 营业收入 / 总资产    → 运营效率
  权益乘数 = 总资产 / 净资产         → 财务杠杆
```

### 3.2 杜邦分析框架

```
                    ROE
                     │
        ┌────────────┼────────────┐
        │            │            │
     净利率      资产周转率    权益乘数
        │            │            │
     净利润       营业收入      总资产
     营业收入      总资产       净资产
        │            │            │
     营业收入     营业收入      总负债
     营业成本     总资产       净资产
     期间费用                   │
     所得税                    │
                            资产负债率
```

### 3.3 杜邦分析应用

**通过杜邦分析可以判断 ROE 的驱动因素**:

| ROE 构成 | 高 ROE 来源 | 企业特征 | 风险点 |
|----------|-------------|----------|--------|
| 高净利率 | 产品定价权 | 品牌、技术优势 | 竞争加剧导致降价 |
| 高周转率 | 资产效率高 | 轻资产、高周转 | 行业景气下滑 |
| 高杠杆 | 财务杠杆 | 高负债经营 | 债务风险、利息压力 |

**优秀企业的特征**: 高净利率 + 适度周转 + 适度杠杆

## 四、现金流分析法

### 4.1 现金流质量判断

| 经营现金流 | 净利润 | 判断 | 分析 |
|------------|--------|------|------|
| 正 | 正 | ✅ 健康 | 盈利有现金支撑 |
| 正 | 负 | ⚠️ 扭亏期 | 现金流改善但未盈利 |
| 负 | 正 | ⚠️ 需警惕 | 利润含金量低 |
| 负 | 负 | ❌ 困难 | 经营困难 |

### 4.2 自由现金流分析

**FCF 持续为正的企业特征**:
- 有能力分红
- 有能力回购股票
- 有能力进行战略投资
- 财务状况稳健

**FCF 持续为负的原因**:
- 高速扩张期（需区分是否有效）
- 资本支出过大
- 经营效率低下

### 4.3 现金流操纵识别

**警惕信号**:
1. 净利润持续增长，经营现金流不增长
2. 应收账款增速 > 营收增速
3. 存货增速 > 营收增速
4. 经营现金流/净利润 < 0.5 持续多年

## 五、行业分析要点

### 5.1 建筑行业特点

**行业特征**:
- 高负债运营（资产负债率 70-80% 为常态）
- 低毛利率（5-15%）
- 规模驱动（订单量决定收入）
- 项目周期长（收入确认滞后）
- 现金流波动大（回款周期不均）

**关键关注点**:
1. 新签订单量及增速
2. 在手订单/营收比（订单储备）
3. 应收账款账龄结构
4. 经营现金流与净利润匹配度
5. 有息负债成本

## 六、健康评分标准

### 6.1 五维度评分体系 (100分制)

| 维度 | 权重 | 指标 | 满分条件 | 零分条件 |
|------|------|------|----------|----------|
| **盈利能力** | 25% | ROE | ROE > 15% | ROE < 0% |
| **偿债能力** | 25% | 资产负债率 | < 40% | > 80% |
| **运营效率** | 20% | 资产周转率 | > 1.0 | < 0.3 |
| **成长能力** | 15% | 营收增长率 | > 20% | < -10% |
| **现金流质量** | 15% | 现金流/净利润 | > 1.2 | < 0.5 |

### 6.2 评分计算公式

```python
def calculate_health_score(metrics):
    score = 0

    # 盈利能力 (25分)
    roe = metrics.get('roe', 0)
    if roe > 15:
        score += 25
    elif roe > 10:
        score += 20
    elif roe > 5:
        score += 15
    elif roe > 0:
        score += 10

    # 偿债能力 (25分)
    debt_ratio = metrics.get('debt_ratio', 100)
    if debt_ratio < 40:
        score += 25
    elif debt_ratio < 50:
        score += 20
    elif debt_ratio < 60:
        score += 15
    elif debt_ratio < 70:
        score += 10
    else:
        score += 5

    # 运营效率 (20分)
    turnover = metrics.get('asset_turnover', 0)
    if turnover > 1.0:
        score += 20
    elif turnover > 0.7:
        score += 15
    elif turnover > 0.5:
        score += 10
    elif turnover > 0.3:
        score += 5

    # 成长能力 (15分)
    growth = metrics.get('revenue_growth', 0)
    if growth > 20:
        score += 15
    elif growth > 10:
        score += 12
    elif growth > 0:
        score += 8
    elif growth > -10:
        score += 4

    # 现金流质量 (15分)
    ocf_ratio = metrics.get('ocf_to_np', 0)
    if ocf_ratio > 1.2:
        score += 15
    elif ocf_ratio > 1.0:
        score += 12
    elif ocf_ratio > 0.8:
        score += 8
    elif ocf_ratio > 0.5:
        score += 4

    return score
```

### 6.3 风险等级判定

| 评分 | 风险等级 | 投资建议 |
|------|----------|----------|
| 80-100 | 低风险 ✅ | 可考虑配置 |
| 60-79 | 中低风险 | 适度配置 |
| 40-59 | 中等风险 ⚠️ | 谨慎配置 |
| 20-39 | 中高风险 | 不建议重仓 |
| 0-19 | 高风险 ❌ | 回避 |

## 八、注意事项

### 8.1 数据使用注意
1. AKShare 数据仅供学术研究，不构成投资建议
2. 数据可能存在延迟或错误，需交叉验证
3. 财务数据需结合宏观经济和行业周期分析

### 8.2 分析局限
1. 历史数据不能预测未来
2. 财务报表可能存在粉饰
3. 单一指标不能说明问题，需综合分析
4. 行业差异大，不能简单跨行业对比

### 8.3 补充分析维度
- 管理层分析（治理结构、股权激励）
- 竞争格局分析（行业集中度、竞争壁垒）
- 政策影响分析（监管政策、产业政策）
- 宏观经济分析（经济周期、利率环境）
